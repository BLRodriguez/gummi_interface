What you need to build the GummiArm

The following tutorial is based on the equipment we have currently in the laboratory. You can of course build one even if you don't have exactly the same configuration (screws, platform ...) but some crucials points need to be respected if you want to use the GummiArm thanks to the code provided on the GummiArm Github

Required set of tools

Here are all the tools we used to mount the soft robot arm :

Our "toolbox" is composed of :

  • Cross-head screwerdrivers
    • H4 X 75
    • PH00 X 50

    • 2.5 mm
    • 2.0 mm

Required non-plastic pieces

As explained before, we encourage you to appropriate the GummiArm ! But you if you want to build one, you need these pieces :

Power supplier settled at 12V and 0.1A

SMPS2Dynamixel (X2)

USB2Dynamixel (X2)

Dynamixel AX/MX 3Pin Bus Hub (X3)

140mm Slewing Bearing (X2) (Trossen Robotics)

120mm Slewing Bearing (Trossen Robotics)

Microsoft Kinect (or other camera device)

Dynamixel 60mm 3 Pin cable

Dynamixel 100mm 3 Pin cable

Dynamixel 140mm 3 Pin cable

Dynamixel 180mm 3 Pin cable

Dynamixel 200mm 3 Pin cable

Rigid 1mm ∅ wire

Filaflex 2.85 mm ∅ filament (Recreus)

You can of course modify the diameter and the material of the two last wires (it implies slight changes to somes pieces).
